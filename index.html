Objetivos

  • Estrutura básica em HTML sobre produtos e suas respectivas categorias.
  • Nesta fase não será necessário o uso de CSS ou frameworks
  • Não deve haver JavaScript

Metas

  1. Definição de objetivos e metas;
  2. Criação de dados fictícios para serem utilizados;
  3. Fazer esqueleto inicial do HTML com seus metadados (DOCTYPE e head);
  4. Elemento de navegação entre a página inicial, página dos produtos e página de entrega;
  5. Listagem dos produtos em suas categorias;
  6. Listagem dos tipos modalidades de entrega;
  7. Tela inicial com objetivos e metas.
